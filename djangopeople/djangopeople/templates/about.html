About Django People

{% endblock %} {% block content %}{% load humanize %}

Django People lists {{ total_people|apnumber }} Django developers from around the world, in {{ countries|length|apnumber }} different countries. The aim of the site is to help Django developers find like-minded souls near them, and hopefully kick-start some local meet-ups and user groups.

The site was originally built and open-sourced by Simon Willison and Natalie Downe. The code is available on github, you can post feedback and bug reports on the repository's issues page.

Colophon

Country data and location lookups come from GeoNames, while the maps themselves are powered by the Google Maps API. Country flags and icons are by Mark James; US state flags come from Wikipedia. Profile pictures come from Gravatar.

The site uses django-tagging by Jonathan Buchanan for the skills feature. OpenID support (used by {{ openid_users }} people) depends on python-openid (version 2.2.5) and django-openid.

Deployment is gunicorn, hosted on Heroku.
